## XLSX Playbook

- Sheet 1 should act as a dashboard with headline KPIs, callouts, and charts.
- Put raw data and formulas on separate tabs when the workbook will be audited.
- Freeze header rows, enable filters, and set column widths before finishing.
- Use formulas for derived metrics so the workbook stays editable.
- Add conditional formatting only where it changes decision speed, not everywhere.
